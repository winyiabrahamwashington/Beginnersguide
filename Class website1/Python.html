Getting Started with Python

Python is a high-level, interpreted programming language known for its simplicity and readability. Created by Guido van Rossum and first released in 1991, Python has become one of the most popular programming languages in the world.

Python is used for web development, data science, AI, and more

Python emphasizes code readability with its notable use of significant whitespace and simple syntax.

What You'll Need

  • Python installed on your computer (download from python.org)
  • A text editor or IDE (VS Code, PyCharm, IDLE, etc.)
  • Basic computer skills

Why Learn Python?

Python is a great first language because:

  • Readable Syntax: Python code reads almost like English
  • Versatile: Used in web development, data science, AI, automation, and more
  • Large Community: Extensive support and resources available
  • Cross-platform: Runs on Windows, macOS, and Linux
  • Batteries Included: Comes with a comprehensive standard library

Your First Python Program

Here's a very simple Python program that prints "Hello, World!":

# This is a simple Python program print("Hello, World!")

To run this, save it as hello.py and execute it with python hello.py in your terminal.

Variables and Data Types

Variables are containers for storing data values. In Python, you don't need to declare variables before using them or declare their type. The interpreter automatically determines the type based on the value assigned.

# Variable assignment examples name = "Alice" # String (text) age = 25 # Integer (whole number) height = 5.9 # Float (decimal number) is_student = True # Boolean (True/False) hobbies = ["reading", "coding", "hiking"] # List coordinates = (10.5, 20.3) # Tuple person = {"name": "Alice", "age": 25} # Dictionary

Basic Data Types in Python

Type Description Example
int Integer (whole number) 5, -3, 1000
float Floating point number (decimal) 3.14, -0.001, 2.0
str String (text data) "hello", 'Python'
bool Boolean (True/False) True, False
list Ordered, mutable collection [1, 2, 3], ["a", "b"]
tuple Ordered, immutable collection (1, 2, 3), ("a", "b")
dict Key-value pairs {"name": "Alice", "age": 25}
Python is dynamically typed, meaning you don't need to declare variable types explicitly. The interpreter infers types at runtime.

Type Conversion

You can convert between types using built-in functions:

x = 10 # int y = float(x) # Convert to float (10.0) z = str(x) # Convert to string ("10") text = "123" number = int(text) # Convert string to int (123)

Operators

Python has various operators for performing operations on variables and values.

Arithmetic Operators

# Basic arithmetic x = 10 + 5 # Addition (15) y = 10 - 5 # Subtraction (5) z = 10 * 5 # Multiplication (50) w = 10 / 3 # Division (3.333...) v = 10 // 3 # Floor division (3) u = 10 % 3 # Modulus (1) t = 2 ** 3 # Exponentiation (8)

Comparison Operators

# Comparisons return True or False x == y # Equal to x != y # Not equal to x > y # Greater than x < y # Less than x >= y # Greater than or equal to x <= y # Less than or equal to

Logical Operators

# Logical operators x and y # True if both are true x or y # True if at least one is true not x # True if x is false

Operator Example

# Calculate area of a circle radius = 5 pi = 3.14159 area = pi * (radius ** 2) print(f"The area is {area:.2f}") # Format to 2 decimal places

Control Flow

Control flow statements allow you to control the order in which your code executes.

Conditional Statements (if, elif, else)

# If-elif-else example temperature = 30 if temperature > 30: print("It's hot outside") elif temperature > 20: print("It's warm outside") else: print("It's cool outside") # Shorthand if-else (ternary operator) message = "Welcome back!" if is_student else "Please register" print(message)

Loops (for and while)

# For loop examples fruits = ["apple", "banana", "cherry"] for fruit in fruits: print(fruit) # Loop with index for index, fruit in enumerate(fruits): print(f"Fruit #{index + 1}: {fruit}") # While loop example count = 0 while count < 5: print(f"Count: {count}") count += 1 # Same as count = count + 1
Python uses indentation (whitespace) to define code blocks. Consistent indentation is crucial for your code to work correctly.

Loop Control Statements

# break, continue, and pass for num in range(10): if num == 3: continue # Skip this iteration if num == 7: break # Exit the loop print(num) else: print("Loop completed") # Executes if loop didn't break

Functions

Functions are blocks of reusable code that perform a specific task. They help organize and modularize your code.

# Function definition def greet(name, time_of_day="morning"): """This function greets a person""" return f"Good {time_of_day}, {name}!" # Function call print(greet("Alice")) # Uses default parameter print(greet("Bob", "afternoon")) # Overrides default # Function with multiple return values def calculate(x, y): return x+y, x-y, x*y, x/y sum_result, diff_result, prod_result, div_result = calculate(10, 5)

Function Parameters

  • Positional arguments: Must be in correct positional order
  • Keyword arguments: Identified by parameter names
  • Default arguments: Assume default value if not provided
  • Variable-length arguments: Accept arbitrary number of arguments
# Different parameter types def describe_pet(pet_name, animal_type='dog'): # animal_type has default print(f"I have a {animal_type} named {pet_name}.") # Positional argument describe_pet('Willie') # Uses default animal_type describe_pet('Willie', 'hamster') # Keyword argument describe_pet(animal_type='hamster', pet_name='Harry') # Variable-length arguments def make_pizza(*toppings): print("Making pizza with:") for topping in toppings: print(f"- {topping}")
Always include docstrings (the triple-quoted string right after the function definition) to document what your function does.

Data Structures

Python has several built-in data structures for organizing and storing data efficiently.

Lists

# List operations fruits = ['apple', 'banana', 'cherry'] fruits.append('orange') # Add item fruits.insert(1, 'grape') # Insert at position fruits.remove('banana') # Remove item last = fruits.pop() # Remove and return last item length = len(fruits) # Get length sorted_fruits = sorted(fruits) # Returns new sorted list fruits.sort() # Sorts the list in place # List slicing first_two = fruits[:2] # First two items last_two = fruits[-2:] # Last two items every_other = fruits[::2] # Every other item reversed_list = fruits[::-1] # Reversed list

Dictionaries

# Dictionary operations person = { 'name': 'Alice', 'age': 25, 'occupation': 'Engineer' } # Accessing values name = person['name'] # Access with key age = person.get('age') # Safer access with .get() # Modifying person['age'] = 26 # Update value person['city'] = 'New York' # Add new key-value pair del person['occupation'] # Remove key-value pair # Dictionary methods keys = person.keys() # All keys values = person.values() # All values items = person.items() # All key-value pairs

Tuples and Sets

# Tuples are immutable coordinates = (10.5, 20.3) x, y = coordinates # Tuple unpacking # Sets store unique elements unique_numbers = {1, 2, 3, 2, 1} # Becomes {1, 2, 3} unique_numbers.add(4) # Add element unique_numbers.remove(2) # Remove element

Shopping List Example

# Shopping list program shopping_list = [] while True: print("\nCurrent shopping list:", shopping_list) action = input("Add, remove, or quit? ").lower() if action == 'quit': break elif action == 'add': item = input("What to add? ") shopping_list.append(item) elif action == 'remove': item = input("What to remove? ") if item in shopping_list: shopping_list.remove(item) else: print(f"{item} not in list!") else: print("Invalid option!")

This simple program demonstrates lists and basic control flow in a practical example.

File Handling

Python provides built-in functions for working with files, allowing you to read from and write to files on your computer.

# Writing to a file with open("diary.txt", "w") as file: # 'w' for write mode file.write("Today was a good day.\n") file.write("I learned Python file operations.\n") # Reading from a file with open("diary.txt", "r") as file: # 'r' for read mode content = file.read() print(content) # Reading line by line with open("diary.txt", "r") as file: for line in file: print(line.strip()) # strip() removes extra whitespace

Common File Modes

Mode Description
'r' Read (default)
'w' Write (creates new or truncates existing)
'a' Append (creates new or appends to existing)
'r+' Read and write
'b' Binary mode (e.g., 'rb' or 'wb')
Always use the with statement when working with files. It automatically closes the file when done, even if an error occurs.

Working with CSV Files

import csv # Writing to CSV with open('data.csv', 'w', newline='') as file: writer = csv.writer(file) writer.writerow(["Name", "Age", "Occupation"]) writer.writerow(["Alice", 25, "Engineer"]) writer.writerow(["Bob", 30, "Teacher"]) # Reading from CSV with open('data.csv', 'r') as file: reader = csv.reader(file) for row in reader: print(row)

Error Handling

Python uses exceptions to handle errors gracefully. You can catch and handle exceptions to prevent your program from crashing.

# Basic try-except block try: age = int(input("Enter your age: ")) print(f"You are {age} years old") except ValueError: print("Please enter a valid number!") # Handling multiple exceptions try: result = 10 / 0 num = int("text") except ZeroDivisionError: print("Cannot divide by zero!") except ValueError: print("Invalid number conversion!") except Exception as e: print(f"An error occurred: {e}") else: print("No errors occurred!") finally: print("This always runs, with or without errors")

Common Python Exceptions

  • SyntaxError: Invalid syntax
  • IndentationError: Incorrect indentation
  • NameError: Variable not found
  • TypeError: Operation on wrong type
  • ValueError: Invalid value
  • IndexError: List index out of range
  • KeyError: Dictionary key not found
  • FileNotFoundError: File doesn't exist

Age Validation Example

def get_valid_age(): while True: try: age = int(input("Enter your age (0-120): ")) if 0 <= age <= 120: return age else: print("Age must be between 0 and 120") except ValueError: print("Please enter a valid number!") age = get_valid_age() print(f"Your age is {age}")

Object-Oriented Programming

Python supports object-oriented programming (OOP) which allows you to create your own objects with methods and attributes.

# Class definition class Dog: # Class attribute species = "Canis familiaris" # Initializer / Instance attributes def __init__(self, name, age): self.name = name self.age = age # Instance method def description(self): return f"{self.name} is {self.age} years old" # Another instance method def speak(self, sound): return f"{self.name} says {sound}" # Creating objects dog1 = Dog("Buddy", 5) dog2 = Dog("Molly", 3) # Accessing attributes and methods print(dog1.description()) print(dog2.speak("Woof!"))

OOP Concepts

  • Class: Blueprint for creating objects
  • Object: Instance of a class
  • Inheritance: Creating new classes from existing ones
  • Polymorphism: Using a unified interface for different types
  • Encapsulation: Restricting access to certain components
# Inheritance example class Bulldog(Dog): # Inherits from Dog def speak(self, sound="Woof Woof!"): return super().speak(sound) # Call parent method # Polymorphism example def pet_speak(pet): print(pet.speak()) bulldog = Bulldog("Max", 2) pet_speak(bulldog) # Works with any object that has speak()
The self parameter in methods refers to the instance of the class. It must be included in method definitions but is passed automatically when calling the method.

Modules and Packages

Modules are files containing Python code that can be imported into other Python files. Packages are collections of modules.

# Importing modules import math from datetime import date import random as rd from os import path # Using imported modules print(math.sqrt(16)) # 4.0 print(date.today()) # Current date print(rd.randint(1, 10)) # Random number print(path.exists("myfile")) # Check if file exists

Creating Your Own Module

1. Create a file named mymodule.py:

def add(a, b): return a + b def subtract(a, b): return a - b

2. In another file, import and use your module:

import mymodule result = mymodule.add(5, 3) print(result) # Output: 8 # Alternative import from mymodule import subtract print(subtract(10, 4)) # Output: 6

Popular Python Standard Library Modules

  • math: Mathematical functions
  • random: Generate random numbers
  • datetime: Date and time operations
  • os: Operating system interfaces
  • sys: System-specific parameters
  • json: JSON encoder and decoder
  • re: Regular expressions
